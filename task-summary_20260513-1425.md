# HB销量统计 - 一键生成小程序

## 目标
用户需要一个双击即可运行的工具，方便处理不同月份的HB销量数据。

## 交付物
1. **一键运行程序**: ~/Desktop/月度销量分析/HB销量统计_一键生成.command（双击运行）
2. **通用脚本**: ~/.qclaw/workspace/scripts/hb_sales_stats.py（支持命令行参数）

## 功能
- 双击 .command 文件 → 弹出菜单选择数据文件
- 也可将Excel文件拖拽到 .command 图标上直接处理
- 三种输入方式：目录选择/手动路径/拖拽
- 自动检测年月生成输出文件名（HB销量数据202603.xlsx → HB销量统计表_202603.xlsx）
- 处理完成可选自动打开Excel
- 自动安装依赖（openpyxl, pypinyin）

## 测试结果
- 通用脚本 hb_sales_stats.py 成功处理3月数据：36个合作方、260512条、44行
- .command 文件已赋予执行权限（chmod +x）

## 注意
- 首次运行 .command 时macOS可能提示"无法验证开发者"，需右键→打开→确认
- 原 v3 脚本保留不变，新脚本 hb_sales_stats.py 是通用版本
