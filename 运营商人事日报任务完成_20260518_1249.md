## 关键技术点

### 1. 搜索策略
- 使用 `--freshness=7d` 参数限制搜索时间范围为最近7天
- 使用 `--cnt=30` 或 `--cnt=50` 增加返回结果数量
- 针对不同运营商分别搜索，提高召回率

### 2. 腾讯文档表格写入
- 使用 `qclaw_tdoc_mcp_call` 工具（直连HTTP，无长度限制）
- 使用 `sheet.set_range_value` 批量写入（禁止逐个单元格调用）
- 从第100行开始写入，避免覆盖已有数据
- values数组格式：`[{"row": 100, "col": 0, "value_type": "STRING", "string_value": "..."}, ...]`

### 3. PDF生成
- 使用reportlab库，支持中文字体（PingFang.ttc）
- 自定义样式：标题、副标题、正文、分类标题
- 使用Table和TableStyle实现表格排版
- 按日期排序（新到旧）

### 4. 飞书推送
- 使用 `message` 工具发送PDF文件（filePath参数）
- 使用 `feishu_doc` 工具追加内容到文档（append动作）
- Markdown格式内容，支持表格、标题、列表等
