## 五、已知问题与修复记录

### 问题 1：腾讯文档写入步骤缺失（已修复）
- **现象**：prompt 里有"人员归档"的文字描述，但没有实际的工具调用指令
- **原因**：AI 执行任务时只看具体指令，不会主动"理解"文字描述去调用工具
- **修复**：在 prompt 第三步明确写入工具调用指令（`mcporter` 调用 `tencent-docs`）
- **修复时间**：2026-05-18 12:02

### 问题 2：手动触发任务不执行腾讯文档写入
- **现象**：用 `openclaw cron run <id>` 手动触发的任务，没有写入腾讯文档
- **原因**：`cron run` 只是普通运行，不会触发 agent 去执行 prompt 里的所有步骤
- **正确方式**：让 agent **完整执行** prompt（搜索→生成报告→写入腾讯文档→生成 PDF→推送）
- **待验证**：明天（2026-05-19）08:00 自动执行时是否能成功写入

### 问题 3：微信版不需要腾讯文档写入
- **用户确认**：微信版（task ID `14ed4899-ac34-4eb5-99e9-2684ce181426`）只推 PDF 摘要，不需要写入腾讯文档归档
